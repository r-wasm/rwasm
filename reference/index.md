# Package index

## All functions

- [`add_list()`](https://r-wasm.github.io/rwasm/reference/add_list.md) :
  Add one or more packages from a file

- [`add_pkg()`](https://r-wasm.github.io/rwasm/reference/add_pkg.md) :
  Add R package reference(s) to a package repository

- [`add_repo()`](https://r-wasm.github.io/rwasm/reference/add_repo.md) :
  Add all packages from a CRAN-like repository to a package repository

- [`add_tar_index()`](https://r-wasm.github.io/rwasm/reference/add_tar_index.md)
  :

  Add Emscripten virtual filesystem metadata to a given `tar` archive

- [`build()`](https://r-wasm.github.io/rwasm/reference/build.md) : Build
  one or more R packages for WebAssembly

- [`file_packager()`](https://r-wasm.github.io/rwasm/reference/file_packager.md)
  : Create an Emscripten filesystem image

- [`make_library()`](https://r-wasm.github.io/rwasm/reference/make_library.md)
  : Create an R package library

- [`make_vfs_library()`](https://r-wasm.github.io/rwasm/reference/make_vfs_library.md)
  : Create an Emscripten filesystem image of an R package library

- [`make_vfs_repo()`](https://r-wasm.github.io/rwasm/reference/make_vfs_repo.md)
  : Add Emscripten filesystem images to an R package repository

- [`rm_pkg()`](https://r-wasm.github.io/rwasm/reference/rm_pkg.md) :
  Remove R package(s) from a package repository

- [`write_packages()`](https://r-wasm.github.io/rwasm/reference/write_packages.md)
  :

  Write the `PACKAGES` file for a package repository
